# SimpleVulkan

SimpleVulkan is simple vulkan framework.  

## Dependecies

* vulkan 1.0.0+
* glfw 3.2+
* vkcpp <https://developer.nvidia.com/open-source-vulkan-c-api>

## Supported platform

* [x] Linux
* [ ] MacOSX
* [ ] Windows(Visual Studio)
